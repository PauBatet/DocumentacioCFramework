1

Initialize Project

Make the binary executable and generate your project structure in a new directory.

chmod +x ./CFramework
mkdir my_app && cd my_app
./CFramework
2

Run it!

Run the built-in migrations and start the server. Your app will be live on port 8080.

make migrate
make run

Ready to Build

localhost:8080
