Getting Started

Learn how to set up your environment and build your first high-performance web application using CFramework.

System Requirements

Ensure your development environment meets the following specifications before continuing:

  • GCC 9.0 or higher
  • GNU Make 4.1+
  • SQLite3 (for default database engine)
  • Linux or MacOS (Windows via WSL2)

Manual Installation

The core framework is distributed as a pre-compiled binary for project scaffolding. Execute the following in your terminal:

# 1. Provide execution permissions
chmod +x ./CFramework

# 2. Run the binary to generate structure
./CFramework

The generator will create the following directory tree:

.
├── config.c
├── config.h
├── Makefile
├── models.c
├── routes.c
├── templates
│   └── home.html
└── views.c

Starting the Server

Once your structure is generated, use the provided Makefile to manage the application lifecycle.

make migrate && make run

This command compiles the project source, runs pending SQL migrations, and binds the web server to http://localhost:8080.
